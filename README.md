# ESPN 0-Fer Dashboard

Live batting stat dashboard for ESPN college baseball/softball graphics operators.

## Access
**Live URL:** `https://YOUR-USERNAME.github.io/ofer-dashboard/`

Open in **Chrome** for full live XML reading functionality.

## How to use
1. Open the dashboard URL in Chrome
2. Click **📂 BROWSE TO LIVE XML** and select your StatBroadcast XML file
   - e.g. `\\sb-venus-p\xml\2026.xml`
   - Chrome will ask for permission — click **Allow**
3. The dashboard auto-refreshes every 10 seconds from disk
4. Use the **AWAY / HOME color pickers** in the toolbar to set team colors
5. Click **COPY** on any player card to copy their stat line for the graphics system

## Features
- Escalator batting order — current batter always at top
- ESPN 0-fer guide compliant stat lines
- TODAY / TONIGHT label based on game start time
- BOTH TEAMS / AWAY / HOME tab toggle
- Season stats (BA · HR · RBI) pulled from XML

## Updating the dashboard
Replace `index.html` with a new version and push to GitHub — the URL stays the same.
